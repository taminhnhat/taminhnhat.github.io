MURIN

It's name inspired by an isopod "murina cubaris" . Murin is a customizable mobile robotic platform, designed to accelerate robotics research.

FEATURES

  • Powerd by ROS2, Nodejs and Websocket
  • Navigation using Intel Realsense and lidar
  • Support Omni-directional drive and differential drive
  • Rich in user interface: dashboard, game controller, WebSocket API
Murin API Murin ROS2

TECH SPECS

MAX SPEED 1.5 m/s
MAX PAYLOAD 4 kg
POWER 207 Wh (4 hours at normal operation)
SENSOR Bno085 9-dof Orientation IMU Fusion
ld19 Lidar with 12m range
Wattmeter monitoring power in real time
Realsense D435 depth camera
USER INTERFACE Dashboard (Powered by Nodejs and WebSocket)
Manual Mode (gamepad support or onscreen controller)
